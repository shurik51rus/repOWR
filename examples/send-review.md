# Как отправить отзыв

Отзыв — это обычная транзакция SPW токена с комментарием в формате repOWR.

## Что нужно

1. TON-кошелёк (Tonkeeper, MyTonWallet или другой)
2. Токены SPW (купить на [STON.fi](https://app.ston.fi))
3. Адрес кошелька получателя

## Пошагово

1. Откройте кошелёк
2. Выберите токен **SPW**
3. Укажите адрес кошелька получателя
4. Введите любую сумму SPW
5. В поле **комментарий** напишите отзыв в одном из форматов ниже
6. Отправьте транзакцию

Отзыв появится в системе автоматически после подтверждения транзакции.

## Упрощённый формат

Синтаксис: `repOWR:ОЦЕНКА:КОММЕНТАРИЙ:`

Оценка — число от 1 до 5. Комментарий — необязательный, до 500 символов.

### Примеры

```
repOWR:5:
```
Оценка 5 без комментария.

```
repOWR:4:Хорошая работа, быстро и качественно:
```
Оценка 4 с комментарием.

```
repOWR:2:Не выполнил условия сделки:
```
Негативный отзыв.

**Важно:** сообщение должно заканчиваться двоеточием `:`.

## JSON формат

Для детальных отзывов с типом сделки и ссылкой-доказательством.

### Минимальный
```json
{"protocol":"repOWR","rating":5}
```

### С комментарием
```json
{"protocol":"repOWR","rating":5,"comment":"Быстрая сделка, рекомендую"}
```

### Полный
```json
{
  "protocol": "repOWR",
  "rating": 5,
  "type": "deal",
  "comment": "Быстрая P2P сделка, оплата моментально",
  "link": "https://t.me/chat/123"
}
```

## Поля JSON

| Поле | Обязательное | Описание |
|------|:---:|-----------|
| `protocol` | ✓ | Всегда `"repOWR"` |
| `rating` | ✓ | Оценка от 1 до 5 |
| `type` | — | Тип: `deal`, `service`, `product`, `general` |
| `comment` | — | Текст отзыва (до 500 символов) |
| `link` | — | Ссылка-доказательство (URL) |
| `ref` | — | Адрес третьей стороны (арбитр, гарант) |

## Сумма транзакции

Ограничений нет. Можно отправить 0.001 SPW с критикой или 100 SPW с благодарностью — сумма показывает вес отзыва.

## Самооценка

Оценки, отправленные самому себе, не учитываются при расчёте репутации.
