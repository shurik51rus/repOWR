# Как создать профиль

Профиль — это identity-транзакция SPW на адрес ОМР. Данные записываются в блокчейн и отображаются в боте, на сайте и в виджетах.

## Что нужно

1. TON-кошелёк (Tonkeeper, MyTonWallet или другой)
2. Токены SPW (купить на [STON.fi](https://app.ston.fi))

## Параметры транзакции

| Параметр | Значение |
|----------|----------|
| **Получатель** | `UQCywBj5RIyKYf1SeLMkmt9gL13pMzCaqORZZ3iFeJyoRaqO` |
| **Токен** | SPW |
| **Сумма** | Любая |
| **Комментарий** | JSON с полями профиля (см. ниже) |

## Минимальный профиль

```json
{"protocol":"repOWR","type":"identity","nickname":"ВашНик","bio":"Ваше описание"}
```

## Полный профиль

```json
{
  "protocol": "repOWR",
  "type": "identity",
  "nickname": "CryptoTrader",
  "bio": "P2P трейдер, 3 года опыта",
  "skills": ["trading", "defi", "analytics"],
  "languages": ["ru", "en"],
  "location": "Moscow",
  "birth_year": 1990,
  "links": {
    "telegram": "@cryptotrader",
    "github": "https://github.com/cryptotrader"
  },
  "avatar": "https://example.com/photo.jpg"
}
```

## Поля профиля

### Обязательные

| Поле | Тип | Описание |
|------|-----|----------|
| `protocol` | string | Всегда `"repOWR"` |
| `type` | string | Всегда `"identity"` |
| `nickname` | string | Имя/ник, до 30 символов |
| `bio` | string | Описание, до 200 символов |

### Опциональные

| Поле | Тип | Описание |
|------|-----|----------|
| `skills` | array | Навыки, до 10 элементов по 30 символов |
| `languages` | array | Языки, до 10 элементов |
| `location` | string | Город/страна |
| `birth_year` | int | Год рождения (1900–2020) |
| `links` | object | Ссылки на соцсети |
| `avatar` | string | URL фото, до 200 символов |

## Обновление профиля

Отправьте новую identity-транзакцию с обновлёнными данными с **того же кошелька**. Система сохранит последнюю версию. Предыдущие версии остаются в блокчейне, но не используются.

## Безопасность

- **Не указывайте** ФИО, телефон, точный адрес, email
- Рекомендуется создать **отдельный кошелёк** для репутации, не связанный с основными средствами
- Помните: данные в блокчейне **нельзя удалить**
- Ник **не уникальный** — как тёзки в реальной жизни, идентификация по адресу кошелька
